# ⏰ Digital Clock (Python Tkinter)

A simple **Digital Clock GUI application** built using **Python** and **Tkinter**.
This application displays the **current time and date** and updates automatically every second.

---

## 📌 Features

* Real-time **digital clock**
* Displays **hours, minutes, and seconds**
* Shows **AM/PM format**
* Displays **current date**
* Updates **every second automatically**
* Simple and lightweight **Tkinter GUI**

---

## 🖥️ Preview

Example Output:

```
12:48:30 AM
06/03/2026
```
